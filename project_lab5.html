Laboratory 5 | Guess My Birthday Game

This is a Guess My Birthday Game, a game in which the computer will have to guess your number.

Let's say your birthday is 7, In every round out of 5 you will choose that number.

In the first round there is number 7, so I will choose YES between yes/no.

After clicking one of the two buttons it will proceed to the next set of round which is set 2.

In the second round, there is also number 7 so I will choose YES again.

Proceeding to set 3.

This is third round, another number 7 is included in set 3 so I will choose YES.

Proceeding to set 4.

In this round, you will see that there is no number 7 so I will choose the opposite which is NO.

Proceeding to set 5.

There is also no number 7 in this round so I will choose NO.

After those 5 rounds the result will show, the computer will have to guess the number 7.

Tada! The computer guessed it right! The number you had chosen is number 7.
